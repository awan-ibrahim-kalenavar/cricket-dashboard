# Cricket Icons

This folder contains cricket-related icons for the dashboard application.

## Available Icons

- `bat.png` - Cricket bat icon
- `ball.png` - Cricket ball icon

## Usage

Import icons in your components like this:

```javascript
import batIcon from '../assets/icons/bat.png';

// Use in your component
<img src={batIcon} alt="Cricket Bat" width="24" height="24" />
```

## Icon Specifications

- Format: PNG
- Recommended size: 24x24px for standard icons
- Transparent background preferred
- Consistent color scheme

## Adding New Icons

1. Add icon files to this folder
2. Use appropriate naming conventions (lowercase, hyphens for spaces)
3. Maintain consistent sizing and style
4. Update this README file with the new icon information
