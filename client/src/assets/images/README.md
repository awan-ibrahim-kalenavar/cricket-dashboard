# Cricket Images

This folder contains cricket-related images for the dashboard application.

## Available Images

- `india.png` - Indian cricket team flag/logo
- `australia.png` - Australian cricket team flag/logo  
- `england.png` - England cricket team flag/logo

## Usage

Import images in your components like this:

```javascript
import indiaFlag from '../assets/images/india.png';

// Use in your component
<img src={indiaFlag} alt="India" width="32" height="32" />
```

## Adding New Images

1. Add image files to this folder
2. Use appropriate naming conventions (lowercase, hyphens for spaces)
3. Update this README file with the new image information
